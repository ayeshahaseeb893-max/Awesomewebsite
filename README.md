# AwesomeWebsite

**AwesomeWebsite** is a modern, single-page web application built to showcase frontend development skills. It’s fully responsive, visually appealing, and packed with interactive features.

## Features
- Hero section with background video & typing animation
- Responsive navigation (desktop & mobile)
- Login & Signup modals with form validation
- Sections: About, Services, Pricing, Contact
- Interactive pricing cards & service highlights
- Fully functional contact form
- Modern UI/UX with hover effects and responsive grids

## Tech Stack
- HTML, CSS, JavaScript (ES6+)
- Tailwind CSS
- Font Awesome icons
- Responsive design for all devices
